## 常见问题

### 账号与登录

#### 如何获取平台账号？

联系商务业务人员获取账号。可通过试用申请获取体验账号进行产品体验。

#### 支持哪些浏览器登录？

建议使用 **Google Chrome 浏览器** 登录系统，以获得最佳的使用体验。

#### 登录时长有限制吗？

可在系统设置中设置账号登录时长限制，当满足设定时长后将自动退出账号。

---

### 设备管理

#### 支持哪些设备类型？

平台支持多种设备类型，包括前装/后装车载终端、无线定位设备等。具体支持的设备型号可咨询商务人员。

#### 已有设备如何接入平台？

车辆管理平台目前支持**部标标准协议**接入，与各类主流硬件厂家通讯协议兼容。同时针对非联网终端，可提供云管端一体网联方案，为非联网终端提供一站式接入方案。

#### 设备状态显示异常怎么办？

1. 检查设备是否在线
2. 检查设备 SIM 卡流量是否充足
3. 在设备状态页面查看设备历史状态与实时状态
4. 联系技术支持排查

#### 如何批量管理设备？

在设备列表页面，支持以下批量操作：
- 批量导入设备（下载模板填写后导入）
- 批量分配设备到企业/车队
- 批量下发设备指令

---

### 企业管理

#### 一个账号可以管理多个企业吗？

可以。在账号权限配置时，可授予管理多个企业的权限。

#### 如何迁移企业归属？

在企业管理页面，查询结果的操作列点击【移动】，选择目标归属后完成迁移。

---

### 车辆监控

#### 什么设备支持视频直播和回放？

凡是安装**智能网联视频终端**的车辆，通讯协议为部标标准协议或麦谷车辆设备均支持视频远程调阅。

#### 视频回放支持下载吗？

支持。在视频回放的文件页面，点击下载图标可提取视频到本地电脑。

#### 轨迹回放数据保留多久？

轨迹数据保留时长根据平台配置而定，具体可咨询商务或技术支持。

#### 如何快速查找某地址周围的车辆？

使用**区域查车**功能：
1. 进入车辆监控 → 定位/视频页面
2. 在右上角搜索框输入查询地址
3. 点击工具 → 区域查车，即可显示目标地址周围的车辆

#### 电子围栏支持哪些类型？

目前支持以下围栏类型：
- 区域围栏
- 自定义围栏
- 行政区域围栏
- 分段限速围栏
- 线路关键围栏
- 偏离线路围栏

### 统计报表

#### 报表支持导出吗？

支持。所有统计报表和报警报表均支持导出为 Excel 格式。

#### 如何导出报表？

1. 在报表页面点击左下角【导出】
2. 确认导出提示
3. 等待导出完成提示
4. 在页面右上角下载图标处下载文件

#### 出省报警如何区分未归/已归？

在出省报警页面，通过出省状态筛选，可以清晰查看当前企业内出省未归或出省已归车辆情况。

---

### 车务管理

#### 司机档案支持批量导入吗？

支持。在司机管理页面点击【导入】，下载模板填写后导入。

#### 车辆档案如何批量创建？

在车辆档案页面点击【导入】，下载数据导入模板，按照模板填好相关资料后导入。

#### 油费报表能分析哪些数据？

油费报表包含以下分析维度：
- **油费总览**：加油情况、油耗使用情况、加油异常车辆分布
- **油费统计**：每公里油费、百公里油耗
- **油费分析**：每月总油费变化趋势

### 主动安全

#### 主动安全报警包含哪些类型？

主动安全报警包含以下几类：

| 类型 | 报警项 |
|------|--------|
| ADAS（高级辅助驾驶） | 车距过近、行人碰撞、频繁变道、道路标识超限、障碍物、前向碰撞、车道偏离、主动抓拍 |
| DMS（驾驶状态检测） | 疲劳驾驶、接打电话、抽烟、分神驾驶、驾驶员异常、自动抓拍、驾驶员变更 |
| BSD（盲区检测） | 后方接近、左侧后方接近、右侧后方接近 |
| 激烈驾驶 | 急加速、急减速、急转弯、怠速、异常熄火、空挡滑行、发动机超转 |

#### 如何查看主动安全事件附件？

1. 选择相应的主动安全报表
2. 通过事件、报警级别、设备名称等筛选数据
3. 点击操作【查看】，查看对应报警事件附件
4. 也可批量选择附件查看

### 系统管理

#### 操作日志保留多久？

操作日志保留时长根据平台配置而定，可在操作日志页面通过操作时间、登录账号、设备名称、IMEI 等条件查询。

#### 如何自定义首页布局？

通过长按首页数据项，可自由灵活移动拖拽数据布局。

#### 视频直播/回放可以设置自动关闭吗？

可以。在系统设置中可设置视频直播/回放的自动关闭时长，满足设定时长后自动关闭。

### API 接入

#### 已有业务平台如何接入？

提供一站式 API 接口服务能力，开放平台能力 API 分为三类：

| 类型 | 说明 |
|------|------|
| 应用接入 API | 支持平台应用和移动端 APP 应用接入麦谷车联网 PaaS 平台，快速开发自己的服务端应用和用户端应用 |
| 设备接入 API | 支持设备接入到麦谷的设备接入平台，使自有设备获得快速上云的能力 |
| AI 开放能力 API | 可使用麦谷在车联网 AIoT 平台独有的 AI 能力，开发基于 AI 的特色应用 |

具体 API 文档请联系技术支持获取。
