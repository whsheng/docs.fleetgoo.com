## 功能介绍

### 设备管理

- 设备列表管理（新增、导入、分配、修改）
- 设备状态监控（电量统计、历史状态、实时状态）
- 设备指令批量下发
- 移动设备分配

### 企业/组织管理

- 企业创建与管理
- 企业移动、编辑、删除

### 角色/账号/权限管理

- 角色创建与权限配置
- 账号创建、编辑、删除

### 首页看板

- 数据项添加
- 页面布局自定义（支持拖拽）

### 报表统计

#### 业务报表

| 报表类型 | 说明 |
|----------|------|
| 里程报表 | 统计车辆行驶里程数据 |
| 停留报表 | 统计车辆停留时长数据 |
| 离线报表 | 统计设备离线数据 |
| 行程分析 | 分析车辆行程数据 |
| 报警处置 | 统计报警处置情况 |

#### 报警报表

| 报表类型 | 说明 |
|----------|------|
| 主动安全报表 | ADAS、DMS、BSD 等主动安全报警统计 |
| 历史报警 | 历史报警记录查询 |
| 超速报警 | 超速报警统计 |
| 出省报警 | 出省报警统计（支持出省未归/已归筛选） |
| 越界报警 | 越界报警统计 |
| 疲劳驾驶报警 | 疲劳驾驶报警统计 |
| 视频信号遮挡 | 视频信号遮挡报警统计 |
| 驾驶员异常 | 驾驶员异常行为统计 |
| 接打电话报警 | 接打电话报警统计 |

### 车务管理

| 功能模块 | 说明 |
|----------|------|
| 年检提醒 | 车辆年检记录查询与提醒 |
| 保养提醒 | 车辆保养记录查询与提醒 |
| 保险提醒 | 车辆保险记录查询与提醒 |
| 司机档案 | 司机信息管理、驾驶评分 |
| 车辆档案 | 车辆信息管理（单车/批量创建） |
| 油卡管理 | 油卡信息管理 |
| 油费报表 | 油费明细导入、油费总览、油费统计、油费分析 |

### 车辆监控

| 功能模块 | 说明 |
|----------|------|
| 视频直播 | 实时查看车内视频（支持快捷入口） |
| 视频回放 | 历史视频调阅、下载 |
| 实时监控 | 车辆实时位置监控、区域查车、自定义字段展示 |
| 轨迹回放 | 轨迹统计、行程数据、多媒体/事件、轨迹明细导出 |
| 电子围栏 | 区域围栏、自定义围栏、行政区域围栏、分段限速围栏、线路关键围栏、偏离线路围栏 |

### 主动安全

平台支持对以下主动安全事件预警、处理：

#### 高级辅助驾驶（ADAS）

- 车距过近报警
- 行人碰撞报警
- 频繁变道报警
- 道路标识超限报警
- 障碍物报警
- 前向碰撞报警
- 车道偏离报警
- 主动抓拍事件

#### 驾驶状态检测（DMS）

- 疲劳驾驶报警
- 接打电话报警
- 抽烟报警
- 分神驾驶报警
- 驾驶员异常报警
- 自动抓拍事件
- 驾驶员变更事件

#### 盲区检测（BSD）

- 后方接近报警
- 左侧后方接近报警
- 右侧后方接近报警

#### 激烈驾驶报警

- 急加速报警
- 急减速报警
- 急转弯报警
- 怠速报警
- 异常熄火报警
- 空挡滑行报警
- 发动机超转报警

### 系统管理

- 操作日志查询
- 系统设置（登录时长限制、默认首页、视频直播/回放设置）
- 版本信息查看
