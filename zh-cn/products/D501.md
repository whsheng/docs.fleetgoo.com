# D501 产品手册

> D501车联网记录仪是一款高性能汽车级设备，专为车队管理和车辆监控设计。它集成了先进的ADAS/DMS算法，支持多通道云视频存储、4G/5G通信和厘米级高精度定位，为各类商用车辆应用提供卓越的安全性和运营效率。

## 1. 产品概述

### 1.1 产品简介

D501车联网记录仪专为满足各行业的严苛需求而设计。采用专业高性能汽车级图像处理芯片和Linux操作系统，能在-20°C至70°C的工业环境中稳定运行。它集成了先进的驾驶辅助系统(ADAS)和驾驶员监控系统(DMS)。利用AI变帧技术，D501支持多通道视频的全面云存储。同时具备4G/5G全网通信和RTK厘米级高精度定位功能。

### 1.2 核心特点

- 支持2K + 两路1080p视频流
- 汽车级芯片，工业级-20°C至70°C工作温度范围
- AI变帧视频云存储技术，实现多通道视频全面云存储
- 4G/5G全网通信
- 北斗/GPS/RTK厘米级高精度定位
- 集成ADAS/DMS/BSD算法
- 支持远程和自动ADAS校准、实时故障跟踪、远程参数配置等
- 符合部标794、808等相关协议标准
- 支持车载9-36V宽电压输入，具备完善的电路保护(欠压、短路、反接)

## 3. 安装说明

为确保安装顺利、高效、完整，请按以下步骤操作：

- 确定最佳接线点，具体定位车辆保险盒和合适的接地点
- 确定D501记录仪和其他摄像头的理想安装位置
- 规划接线策略：与车辆现有线路集成或选择单独的隐蔽接线
- 根据上述条件，从行车记录仪安装位置开始接线
- 接线完成并确认通电后，将安装图片和相关信息发送给后台人员(如适用)进行数据验证。确认后，重新组装车辆装饰板，牢固安装行车记录仪，完成安装

### 3.1 安装步骤

**安装注意事项：**

- 确保安装牢固防水。避免高温区域和磁干扰源(如车载CD播放器、音响扬声器、车载电脑、车载收音机)
- 确保挡风玻璃上的安装位置在雨刷范围内，以保持清晰视野，即使在雨天也是如此。应靠近后视镜放置，以获得最佳视野
- 不要用手指触摸镜头，因为油脂会留下污迹，导致视频模糊或图像失真
- 安装过程中，确保所有插头接口正确牢固连接。为了增加保护，用电工胶带包裹连接处，防止进水、氧化和意外脱落。确保接线隐蔽，避免阻挡驾驶员视线并保持美观

#### 步骤1：插入卡片

将TF卡和SIM卡插入设备，然后固定卡盖。

拆除安装区域的装饰板。布线电源线束时，按照常见车型的指示沿A柱引导。

设置好前置摄像头角度后，通过拧紧其盖板下的螺丝来固定。这可防止车辆行驶过程中的振动或意外调整镜头方向。

![固定前置摄像头](../../../en-us/images/d501-camera-securing.png)
*图10：固定前置摄像头*

#### 步骤5：DMS摄像头安装

DMS摄像头安装位置应遵循以下原则：

- **安装位置：** 建议安装在中控台或仪表板上
- **安装角度和距离：** 确保驾驶员在摄像头正前方±30°范围内。建议角度应尽可能小。摄像头与驾驶员面部之间的建议距离为60-120cm，理想情况下约80cm
- **面向居中：** 确保驾驶员面部在DMS摄像头视野中居中(可通过手机APP确认)
- **无遮挡：** 确保DMS摄像头不会阻挡驾驶员视线或干扰驾驶
- **清晰视野：** 确保没有物体(如方向盘)阻挡DMS摄像头与驾驶员面部之间的视线
- **水平对齐：** DMS摄像头应水平安装，不得倾斜
- **最佳角度：** 在上述条件下，DMS摄像头与驾驶员面部之间的偏差角度越小越好。理想情况下，应直接指向驾驶员

*图11：DMS摄像头安装指南(如实际情况不同，选择合理的安装和固定位置)*

### 3.2 接线说明

将电源线布线到保险盒。将ACC线和常电线连接到相应的保险槽。将GND线直接连接到合适的接地点(如车辆底盘上的金属螺栓)。

![接线图](../../../en-us/images/d501-wiring-diagram.png)
*图12：接线图*


| 编号 | 线色 | 描述 |
| :-- | :-- | :-- |
| 1 | 黄色 | 电源正极(B+) |
| 2 | 黑色 | 电源负极(GND) |
| 3 | 红色 | ACC(附件/点火开关控制) |

**接线注意事项：**

- 拆除车辆原有对应保险丝，用电源线的保险丝替换
- 设备通常配备15A保险丝。不要用低于15A的保险丝替换
- ACC线(红色)控制设备的休眠/唤醒状态。不要将ACC线连接到常电源
- 连接到正极(B+)的常电线在点火关闭时应保持至少12V

## 5. 常见问题(FAQ)


| 问题/故障 | 解决方案 |
| :---- | :---- |
| 设备脱落 | 确保车玻璃彻底清洁，3M胶带牢固按压。如有必要，更换3M胶。 |
| 记录仪不开机 | 请确认主电源线(如宝马接口)插入牢固，电源线连接正确，确保外部电源正常工作。 |
| 开机后视频录制不启动 | 首先确保TF卡正确插入。如果没有，请在设备断电后重新插入TF卡。如果这些步骤失败，请格式化TF卡或更换新的TF卡。 |
| 录制异常终止 | 请格式化TF卡或更换符合要求的TF卡(Class 10或更高)。 |
| 视频画面模糊 | 请确保车辆挡风玻璃上没有污渍。<br>请确保记录仪摄像头镜头上没有污渍或遮挡物。 |
| 记录仪无响应 | 请断开电源并重启记录仪。或者，按设备的复位按钮自动重启记录仪。 |
| 终端离线 | 检查SIM卡是否欠费；如果是，请联系网络运营商结算账单。<br>检查SIM卡是否接触良好：重新插入SIM卡。<br>如果车辆在信号较弱的区域，如地下停车场或隧道，请移动到信号接收更好的区域。 |
| 终端不定位 | 确保天线连接接口插入牢固。<br>检查天线表面是否朝上：重新调整天线位置。<br>如果车辆在地下停车场、隧道或其他无信号区域，请驶出该区域。 |
