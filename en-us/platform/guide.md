#### What Data Does Trajectory Playback Support Displaying

**Trajectory Statistics**

Click [Statistics] in the information bar below the map. Linear charts are formed based on trajectories within the query time period, displaying relevant statistical data.

![](http://mapgoo-oms.oss-cn-shenzhen.aliyuncs.com/1695351732_image.png)

### Electronic Fence

#### How to Open Electronic Fence

In the menu bar, **Vehicle Monitoring** → **Electronic Fence** to enter the electronic fence page.

![](http://mapgoo-oms.oss-cn-shenzhen.aliyuncs.com/1695603528_image.png)

#### Which Fence Types Are Supported

Currently supports area fence, custom fence, administrative region fence, segmented speed limit fence, route key fence, and route deviation fence.

#### How to Create an Area Fence

**Step 1**: Open **Electronic Fence** → **Area Fence**. Enter the license plate number or fence name in the search box to query. Click Add in the upper right corner of the page to create a fence. You can bind and edit created fences.

![](http://mapgoo-oms.oss-cn-shenzhen.aliyuncs.com/1695351799_image.png)

**Step 2**: Create a custom fence or administrative region fence.

According to your needs, define the fence range through circles or polygons. In the upper right corner of the map on the add page, click Search to search for the corresponding address. Then select the shape and draw the fence range on the map according to the shape.

![](http://mapgoo-oms.oss-cn-shenzhen.aliyuncs.com/1695351802_image.png)

**Step 3**: Fence Condition Settings.

According to your needs, select entry/exit fence alarm conditions. Click Vehicle Binding to enter the associated vehicle page. You can select vehicles to bind as needed. After binding the vehicles, completion is achieved.

![](http://mapgoo-oms.oss-cn-shenzhen.aliyuncs.com/1695351810_image.png)

### System Settings

#### How to Open System Settings

Click my avatar in the upper right corner. A drop-down options will appear. Click System Settings.

![](http://mapgoo-oms.oss-cn-shenzhen.aliyuncs.com/1695608576_image.png)

#### What Content Can Be Set

- **Basic Settings**: You can set the current account login time limit. When the duration is reached, the account will automatically log out. The default homepage can be set to select the homepage interface when the user logs in.
- **Live Video**: You can set the automatic close duration when clicking live video. The system will automatically close after the set duration is reached.
- **Video Playback**: You can set the automatic close duration when clicking video playback. The system will automatically close after the set duration is reached.
