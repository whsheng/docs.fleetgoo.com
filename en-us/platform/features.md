## Features

### Device Management

- Device list management (add, import, assign, modify)
- Device status monitoring (battery statistics, historical status, real-time status)
- Batch device command issuance
- Mobile device assignment

### Enterprise/Organization Management

- Enterprise creation and management
- Enterprise move, edit, delete

### Role/Account/Permission Management

- Role creation and permission configuration
- Account creation, edit, delete

### Dashboard

- Data item addition
- Custom page layout (drag-and-drop support)

### Statistical Reports

#### Business Reports

| Report Type | Description |
|----------|------|
| Mileage Report | Statistics on vehicle mileage data |
| Stop Report | Statistics on vehicle stop duration |
| Offline Report | Statistics on device offline data |
| Trip Analysis | Analysis of vehicle trip data |
| Alarm Disposal | Statistics on alarm disposal status |

#### Alarm Reports

| Report Type | Description |
|----------|------|
| Active Safety Report | ADAS, DMS, BSD and other active safety alarm statistics |
| Historical Alarm | Historical alarm record query |
| Overspeed Alarm | Overspeed alarm statistics |
| Out-of-Province Alarm | Out-of-province alarm statistics (supports filtering by returned/not returned) |
| Boundary Alarm | Boundary alarm statistics |
| Fatigue Driving Alarm | Fatigue driving alarm statistics |
| Video Signal Occlusion | Video signal occlusion alarm statistics |
| Driver Abnormality | Driver abnormal behavior statistics |
| Phone Call Alarm | Phone call while driving alarm statistics |

| Module | Description |

### Active Safety

The platform supports early warning and handling of the following active safety events:

#### Advanced Driver Assistance Systems (ADAS)

- Close distance alarm
- Pedestrian collision alarm
- Frequent lane change alarm
- Road sign exceedance alarm
- Obstacle alarm
- Forward collision alarm
- Lane departure alarm
- Active capture event

#### Driver Monitoring System (DMS)

- Fatigue driving alarm
- Phone call alarm
- Smoking alarm
- Distracted driving alarm
- Driver abnormality alarm
- Auto capture event
- Driver change event

#### Blind Spot Detection (BSD)

- Rear approach alarm
- Left rear approach alarm
- Right rear approach alarm

#### Aggressive Driving Alarm

- Rapid acceleration alarm
- Rapid deceleration alarm
- Sharp turn alarm
- Idle alarm
- Abnormal engine off alarm
- Neutral coasting alarm
- Engine overspeed alarm

### System Management

- Operation log query
- System settings (login duration limit, default homepage, live video/playback settings)
- Version information view
