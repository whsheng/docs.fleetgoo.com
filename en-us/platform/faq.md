## FAQ

### Account & Login

#### How to get a platform account?

Contact the sales business representative to obtain an account. You can get an experience account for product trial through trial application.

#### Which browsers are supported?

It is recommended to use **Google Chrome browser** to log in to the system for the best user experience.

#### Is there a login time limit?

You can set the account login time limit in the system settings. The system will automatically log out when the set time is reached.

---

### Device Management

#### Which device types are supported?

The platform supports multiple device types, including front/rear-mounted vehicle terminals, wireless positioning devices, etc. For specific supported device models, please consult the sales representative.

#### How to connect existing devices to the platform?

The vehicle management platform currently supports **Ministry standard protocol** integration, compatible with communication protocols of various mainstream hardware manufacturers. For non-networked terminals, we can provide an integrated cloud-pipe-terminal solution for one-stop access.

#### What to do if device status shows abnormal?

1. Check if the device is online
2. Check if the device SIM card has sufficient data
3. View device historical status and real-time status on the device status page
4. Contact technical support for troubleshooting

#### How to manage devices in batch?

On the device list page, the following batch operations are supported:
- Batch import devices (download template, fill in and import)
- Batch assign devices to enterprises/fleets
- Batch issue device commands

---

### Enterprise Management

#### Can one account manage multiple enterprises?

Yes. When configuring account permissions, you can grant permissions to manage multiple enterprises.

#### How to migrate enterprise affiliation?

On the enterprise management page, click [Move] in the operation column of the query result, and select the target affiliation to complete the migration.

---

### Vehicle Monitoring

#### Which devices support live video and playback?

All vehicles equipped with **intelligent networked video terminals** support remote video access, with communication protocols being Ministry standard protocol or Maigu vehicle devices.

#### Does video playback support download?

Yes. On the video playback file page, click the download icon to extract the video to your local computer.

#### How long is trajectory playback data retained?

The retention period of trajectory data depends on the platform configuration. For details, please consult the sales or technical support.

#### How to quickly find vehicles around a specific address?

Use the **Area Vehicle Search** function:
1. Go to Vehicle Monitoring → Positioning/Video page
2. Enter the query address in the search box in the upper right corner
3. Click Tools → Area Vehicle Search to display vehicles around the target address

#### Which types of electronic fences are supported?

The following fence types are currently supported:
- Area fence
- Custom fence
- Administrative region fence
- Segmented speed limit fence
- Route key fence
- Route deviation fence

---

### Statistical Reports

#### Do reports support export?

Yes. All statistical reports and alarm reports support export to Excel format.

#### How to export reports?

1. Click [Export] in the lower left corner of the report page
2. Confirm the export prompt
3. Wait for the export completion prompt
4. Download the file from the download icon in the upper right corner of the page

#### How to distinguish between not returned/returned in out-of-province alarms?

On the out-of-province alarm page, you can clearly view the status of vehicles that have left the province and not returned or have returned by filtering by out-of-province status.

---

### Vehicle Affairs Management

#### Does driver profile support batch import?

Yes. On the driver management page, click [Import], download the template, fill it in and import.

#### How to create vehicle profiles in batch?

On the vehicle profile page, click [Import], download the data import template, fill in the relevant information according to the template, and import.

#### What data can the fuel cost report analyze?

The fuel cost report includes the following analysis dimensions:
- **Fuel Cost Overview**: Refueling status, fuel consumption usage, distribution of vehicles with abnormal refueling
- **Fuel Cost Statistics**: Fuel cost per kilometer, fuel consumption per 100 kilometers
- **Fuel Cost Analysis**: Monthly total fuel cost change trend

### Active Safety

#### What types of active safety alarms are included?

Active safety alarms include the following categories:

| Type | Alarm Items |
|------|--------|
| ADAS (Advanced Driver Assistance Systems) | Close distance, Pedestrian collision, Frequent lane change, Road sign exceedance, Obstacle, Forward collision, Lane departure, Active capture |
| DMS (Driver Monitoring System) | Fatigue driving, Phone call, Smoking, Distracted driving, Driver abnormality, Auto capture, Driver change |
| BSD (Blind Spot Detection) | Rear approach, Left rear approach, Right rear approach |
| Aggressive Driving | Rapid acceleration, Rapid deceleration, Sharp turn, Idle, Abnormal engine off, Neutral coasting, Engine overspeed |

#### How to view active safety event attachments?

1. Select the corresponding active safety report
2. Filter data by event, alarm level, device name, etc.
3. Click [View] in the operation column to view the corresponding alarm event attachments
4. You can also select attachments in batch for viewing

### System Management

#### How long are operation logs retained?

The retention period of operation logs depends on the platform configuration. You can query by operation time, login account, device name, IMEI, etc. on the operation log page.

#### How to customize the homepage layout?

You can freely move and drag the data layout by long-pressing the homepage data items.

#### Can live video/playback be set to auto-close?

Yes. You can set the auto-close duration for live video/playback in the system settings. The system will automatically close after the set duration is reached.

### API Integration

#### How to integrate with existing business platforms?

We provide one-stop API interface service capabilities. The open platform capabilities are divided into three categories:

| Type | Description |
|------|------|
| Application Integration API | Supports platform applications and mobile APP applications to access the Maigu IoT PaaS platform, quickly developing your own server-side applications and user-end applications |
| Device Integration API | Supports device integration to the Maigu device access platform, enabling your own devices to have quick cloud access capabilities |
| AI Open Capability API | You can use Maigu's unique AI capabilities on the IoT AIoT platform to develop AI-based featured applications |

For specific API documentation, please contact technical support.
